# CSE341 Assembly Language Projects

This repository contains two Assembly Language projects implemented using Emu8086 as part of the CSE341 course requirements.

## 📌 Projects Included

### 1. Banking System Simulator

A structured banking system that manages multiple accounts, performs transactions, and maintains history using stack operations.

**Key Features:**

* Unique account creation with ID validation
* Secure authentication (ID + PIN)
* Deposit and withdrawal with constraints
* Stack-based transaction history
* Interest calculation using procedures
* Richest account detection

---

### 2. Secure File Access Control System

A security-focused system that simulates user authentication, role-based access control, and system activity tracking.

**Key Features:**

* Multi-user authentication system
* Role-based access (Admin/User)
* Login attempt limitation
* Password validation rules
* Stack-based activity logging
* Account locking mechanism
* Audit log viewer

---

## 🧠 Concepts Covered

* Arrays (data storage)
* Stack (history/logs)
* Procedures (modular design)
* Conditional logic (CMP, Jumps)
* Arithmetic operations (MUL, DIV)

---

## ⚙️ Tools Used

* Emu8086
* x86 Assembly Language

## 📂 How to Run

1. Open Emu8086
2. Navigate to project folder
3. Open `.asm` file
4. Compile and run
