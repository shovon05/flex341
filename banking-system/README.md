# Banking System Simulator

## Overview

This project simulates a banking system using Assembly language. It manages multiple accounts, processes transactions, and maintains transaction history.

## Features

1. Account creation with unique ID validation
2. Secure authentication (ID + PIN)
3. Deposit and withdrawal with validation
4. Stack-based transaction history
5. Interest calculation procedure
6. Richest account finder

## Data Structure

* Array → account records
* Stack → transaction history

## Key Concepts

* CMP and conditional jumps
* Loops and traversal
* Procedures (PROC)
* Arithmetic operations

## How It Works

* Accounts are stored in a fixed array
* Transactions update balances directly in memory
* Stack is used to store transaction history
* Procedures modularize operations
