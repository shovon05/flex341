# Secure File Access Control System

## Overview

This project simulates a secure access control system with authentication, role-based permissions, and activity logging.

## Features

1. Multi-user authentication
2. Role-based access control
3. Login attempt limitation
4. Password validation system
5. Stack-based activity logging
6. Password update feature
7. Account lock system
8. Audit log viewer

## Data Structure

* Array → user records
* Stack → activity logs

## Key Concepts

* String comparison
* Conditional logic
* Stack operations
* Procedures

## How It Works

* Users are stored in an array
* Authentication verifies credentials
* System logs actions using stack
* Admin can review activity logs

## Future Improvements

* Encryption system
